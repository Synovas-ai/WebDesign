# UX / Conversion Notes

- Search bar should work instantly.
- Topic chips should filter content.
- Sidebar CTA should remain visible on desktop.
- Cards should prioritize trust + clarity over clickbait.
- Email signup is secondary to quote CTA.
- Hero image must feel human, practical, believable.
- Entire page should feel more “trusted advisor” than “content farm”.
